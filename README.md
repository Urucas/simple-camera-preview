simple-camera-preview
=====================

A simple html5 code to stream a camera preview

Install
=======
git clone https://github.com/Urucas/simple-camera-preview

npm install

Usage
=====
run ```bash node app.js ```

Open <a href="http://localhost:3000/">http://localhost:3000/</a>

====
